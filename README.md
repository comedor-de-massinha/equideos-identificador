# Identificador de Equinos

Aplicação de classificação de imagem que identifica se um animal é um cavalo, um burro ou uma zebra.

## Como usar

1. Abra `index.html` em um navegador compatível.
2. Clique em **Enviar imagem** e selecione uma foto de um cavalo, burro ou zebra.
3. Aguarde a análise e veja o resultado com a classe e a confiança.

## Estrutura de pastas

- `train/` - pasta com imagens usadas para treinar o modelo.
- `test/` - pasta com imagens usadas para testar o modelo e verificar a precisão.

## Tecnologias

- HTML
- Tailwind CSS
- TensorFlow.js
- Teachable Machine
